# Business Document Types

Use this router before drafting business artifacts.

| Type | Use When | Minimum Artifacts |
| --- | --- | --- |
| `business-plan` | The user is shaping a new business, service, or business model. | `brief.md`, `business-plan.md`, `validation-plan.md`, `issue-candidates.md` |
| `market-entry-analysis` | The user is deciding whether to enter a market, country, customer segment, or channel. | `brief.md`, `assumptions.md`, `source-list.md`, `calculation-model.md`, `market-entry/report.md`, `decision.md`, `validation.md` |
| `profitability-review` | The user needs margin, cost, pricing, CAC, LTV, or break-even review. | `assumptions.md`, `calculation-model.md`, `decision.md`, `validation.md` |
| `pitch-outline` | The user needs investor, partner, or executive presentation material. | `executive-summary.md`, `deck-outline.md`, `source-list.md` |
| `decision-memo` | The user needs a concise go/no-go or sequencing decision. | `decision.md`, `source-list.md`, `issue-candidates.md` |

Prefer the smallest type that answers the decision. Add exports only after Markdown source artifacts are reviewed.
