# Reference Improvements

Project-local backlog for improvement candidates discovered while using reference repositories, templates, or upstream examples.

These records are optional context. They do not expand the active issue scope, block release, or mutate the referenced repository. Promote an entry into `issues/*.md` only after the user chooses to act on it.

## Status Values

| Status | Meaning |
| --- | --- |
| candidate | Captured for later triage |
| accepted | Worth turning into work, but not yet promoted |
| promoted | Converted into a normal ModuFlow issue |
| closed | No longer useful or intentionally declined |
