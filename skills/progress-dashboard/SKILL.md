# Progress Dashboard

Make progress visible without moving source of truth out of Git.

## Dashboard Fields

- issue ID
- title
- phase
- owner
- last updated
- blocker
- next command
- PR/release status

Use `workspace/dashboard.md` as the default low-friction dashboard. A web dashboard can be generated from the same files.
