# Data Analysis Bridge

Every analysis must connect to an issue or roadmap decision.

## Required Fields

- issue ID
- question
- decision supported
- data source
- grain
- period
- filters
- caveats
- recommendation

Save findings to `analysis.md` and metric definitions to `metrics.md`.
