# Roadmap Management

Roadmap is a view over source artifacts.

## Inputs

- Opportunities
- Issues
- Specs
- Metrics and analysis
- Design/prototype readiness
- Dependencies
- Releases

## Output

Use Now/Next/Later.

For every item include:

- issue ID
- outcome
- reason
- confidence
- dependency
- next command
